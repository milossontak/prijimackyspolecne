# Přijímačky Společně

Webová aplikace pro vzdělávací projekt "Přijímačky Společně" - pomoc žákům při přípravě na přijímací zkoušky na střední školy i gymnázia z 5. třídy.

## Rychlý start

```bash
# Instalace závislostí
npm install

# Spuštění vývojového serveru
npm run dev
```

Web poběží na [http://localhost:3000](http://localhost:3000)

### Vytvoření statického webu

```bash
# Build statického webu
npm run build
```

Statické soubory budou v adresáři `out/` - ty můžete nahrát kamkoliv (GitHub Pages, Netlify, Vercel, nebo jakýkoliv webhosting).
